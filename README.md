# MUST视传2026校招情报站 —— 使用说明

## 项目简介
专为**澳门科技大学视觉传达设计2026届本科应届生**打造的校园招聘信息聚合站。
- 聚合粤港澳大湾区、福建省（厦门/福州/泉州）及全国重点城市设计类校招岗位
- 每日自动更新并推送招聘情报
- **福建省重点**：央国企、事业单位、大型上市公司、新特精企业
- 支持按城市、岗位类型、截止时间筛选

## 使用方法

### 1. 浏览网站

**方式A：直接打开**
用浏览器打开 `index.html` 文件即可浏览完整校招信息。

**方式B：启动本地服务器**
```bash
cd 校招情报站
bash start_server.sh
# 然后访问 http://localhost:8080
```

### 2. 生成每日报告

**全国报告：**
```bash
python3 daily_report.py
```

**福建省专属报告（央国企/事业单位/上市公司/新特精）：**
```bash
python3 fujian_daily.py
```

报告将保存到：
- 全国：`reports/report_YYYYMMDD.txt` 和 `reports/latest.txt`
- 福建：`reports/fujian_report_YYYYMMDD.txt` 和 `reports/fujian_latest.txt`

### 3. 自动每日推送（推荐）

已设置系统定时任务：

| 时间 | 任务 | 内容 |
|------|------|------|
| **07:45** | 福建省招聘情报 | 央国企、事业单位、上市公司、新特精企业 |
| **08:45** | 全国校招情报 | 大湾区、北上广深等重点城市 |

查看定时任务：
```bash
crontab -l
```

## GitHub Pages 部署指南

### 第一步：在 GitHub 创建仓库
1. 打开 https://github.com/new
2. 仓库名填 `zhaopin-station`
3. visibility 选 **Public**
4. **不要勾选** "Add a README file"
5. 点击 **Create repository**

### 第二步：推送本地代码到 GitHub
在本地 `校招情报站` 目录下执行：

```bash
# 1. 进入目录
cd 校招情报站

# 2. 添加 GitHub remote（替换 YOUR_USERNAME 为你的GitHub用户名）
git remote add github https://github.com/YOUR_USERNAME/zhaopin-station.git

# 3. 推送到 GitHub
git push -u github master
```

### 第三步：启用 GitHub Pages
1. 打开仓库页面 → 点击 **Settings**
2. 左侧菜单选 **Pages**
3. Source 选 **Deploy from a branch**
4. Branch 选 **master** / **root**
5. 点击 **Save**
6. 等待 1-2 分钟，访问 `https://YOUR_USERNAME.github.io/zhaopin-station`
